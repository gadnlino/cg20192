Clique no canvas para escolher os pontos de controle.

Depois, clique na curva desejada para traçá-la.

Para mover os pontos, clique em Mover pontos e arraste o ponto no canvas.

Para parar de mover os pontos, clique em Parar de mover.

Para apagar o ultimo ponto, clique em Limpar último ponto.

Para apagar a ultimo curva, clique em Limpar última curva.

Para apagar todos os pontos, clique em Limpar todos os pontos.

Para apagar todos as curvas, clique em Limpar todas as curvas.

Para tudo, clique em Limpar tudo.
